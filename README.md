# geomap3D
GeoMap3D is a Python library that makes creating beautiful, interactive 3D geographic maps , without requiring any JavaScript, frontend frameworks, or GIS software.  GeoMap3D lets you generate and export rich 3D maps using a clean, declarative Python API.
